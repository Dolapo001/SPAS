CSV Format Instructions

Required Columns:

  • • matric_no - Student matric number (e.g., SC21001)
  • • cgpa - Student CGPA (e.g., 4.50, 3.75, 2.80)

Example CSV Format:

matric_no,cgpa, email, fullname
SC21001,4.50, johndoe@gmail.com, John Doe
SC21002,3.75, johndoe@gmail.com, John Doe
SC21003,2.80, johndoe@gmail.com, John Doe

Important Notes:

  • • File must be in CSV format (.csv)
  • • Maximum file size: 5MB
  • • First row should contain column headers
  • • Matric numbers must be unique
  • • CGPA values must be between 0.00 and 5.00
